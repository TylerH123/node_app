Node.js

  • Runtime environment built on Chrome's V8 engine
  • Allows you to write server-side applications in Javascript
  • Open source and cross platform

Pros of Using Node Over Python

  • Only need to know one language
  • Npm is one of the largest repositories of software libraries in the world
  • Node apps are faster

Why should we care?

  • According to GitHub, Javascript is the #1 programming language
  • Recently, job searches for Node.js rose by 57%

Simple Server

var http = require('http');
http.createServer(function (req, res) {
    res.writeHead(200, {'Content-Type': 'text/html'});
    res.end('Hello World!');
}).listen(5000);

Modules

  • Set of functions you want to include in your application
  • Similar to modules in Python
  • Use require("[module_name]") to include in your application
  • Each method in the module must have exports. before the name in order to be used in your application

Url Module

  • Helps split the web address into readable parts
  • Using the url.parse() method, we will get an URL object back
  • To get the pathname, we can add .pathname to the end of the method

MongoDB

  • Install with: npm install mongodb
  • Methods include: createCollection, insertOne, findOne, insertMany, deleteOne, dropCollection, and many more

Running Node Apps

To run a node app, simply type node [file_name] in your terminal
